## FOLDER STRUCTURE

```
boardease-site/
├── index.html              ← Home Page
├── data-gathering.html     ← Data Gathering Page
├── results.html            ← Results, Discussion & Recommendation Page
├── css/
│   └── style.css           ← All styles (DO NOT rename)
├── js/
│   └── script.js           ← Navbar, tabs, animations
├── images/                 ← PUT ALL YOUR IMAGES HERE
│   ├── hero-bg.jpg         ← Boarding house background photo
│   ├── boardease-logo.png  ← BoardEase logo (white version)
```

## DEPLOYMENT

Deploy using any of the following (as required):
- **GitHub Pages** — push to a repo, enable Pages in settings
- **Netlify** — drag and drop the `boardease-site/` folder
- **Vercel** — connect your GitHub repo
